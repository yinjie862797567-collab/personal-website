# 个人主页部署指南（30 分钟上线）

> **目标**：把你刚刚生成的 `index.html` 部署到公网，零成本、可访问、专业感。
> **总耗时**：30 分钟

---

## 🚀 推荐方案：**Vercel + GitHub**（最简单、免费、专业）

### 步骤 1：注册 GitHub 账号（5 分钟）

- 访问 https://github.com 注册
- 用户名建议：`jack-captain-yourname` 或类似
- 邮箱验证

### 步骤 2：创建仓库（3 分钟）

1. 登录 GitHub → 右上角 "+" → New repository
2. 仓库名：`jack-captain-site`（或你喜欢的名字）
3. 选 **Public**
4. 勾选 **Add a README file**
5. 点击 **Create repository**

### 步骤 3：上传 index.html（5 分钟）

**方法 A：网页上传（最简单）**

1. 进入新创建的仓库
2. 点击 **Add file** → **Upload files**
3. 把 `index.html` 文件拖进去
4. 点击 **Commit changes**

**方法 B：Git 命令行（更专业）**

```bash
# 在你电脑上打开 PowerShell
cd D:\新建文件夹\智能体编程\04-资源库\personal-website
git init
git add index.html
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/你的用户名/jack-captain-site.git
git push -u origin main
```

### 步骤 4：部署到 Vercel（5 分钟）

1. 访问 https://vercel.com
2. 点击 **Sign Up** → 用 GitHub 账号登录
3. 点击 **Add New...** → **Project**
4. 找到 `jack-captain-site` 仓库 → 点击 **Import**
5. 配置：
   - Framework Preset: **Other**
   - Root Directory: `./`（默认）
6. 点击 **Deploy**

**等待 30 秒，部署完成！**

### 步骤 5：访问你的网站（1 分钟）

- Vercel 会给你一个免费域名：`jack-captain-site.vercel.app`
- 现在全世界都可以访问了 🎉

## ⚠️ 注意事项

1. **不要在网站放联系方式（微信 / 电话）** — 容易被爬虫
   - 改用 "加微信备注 XX" 的引导话术
2. **案例必须是真实脱敏** — 不要瞎编客户
3. **定期更新** — 每月加 1-2 篇案例 / 文章
4. **域名绑定** — 客户主动找时，专业域名更可信
